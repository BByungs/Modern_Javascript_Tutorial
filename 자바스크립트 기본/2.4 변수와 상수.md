# 변수와 상수

변수는 데이터를 저장할때 쓰이는 `이름이 붙은 저장소`다.

> ## 변수 선언 방식

```js
let user = 'John',
  age = 25,
  message = 'Hello';
```

위의 방식은 코드 가독성이 떨어지기 때문에 권장하진 않음.

```js
let user = 'John';
let age = 25;
let message = 'Hello';
```

```js
let user = 'John',
  age = 25,
  message = 'Hello';
```

```js
let user = 'John',
  age = 25,
  message = 'Hello';
```

위의 3가지 코드는 개인의 취향에 따라 사용하면 된다.
